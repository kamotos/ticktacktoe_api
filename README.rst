Basic Commands
--------------

Running tests with py.test
~~~~~~~~~~~~~~~~~~~~~~~~~~

::

  $ docker-compose -f local.yml py.test
  $ # OR
  $ py.test


Running The project
~~~~~~~~~~~~~~~~~~~

::

  $ docker-compose -f local.yml up
